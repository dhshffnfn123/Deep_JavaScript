
# --------------------------------- DataType --------------------------------- #

- 값의 종류를 일컫는다.
- 자바스크립트의 모든 값은 타입을 가진다.
- ES6는 7개의 데이터 타입을 제공한다.
    - Number : 숫자, 정수와 실수 구분 없이 하나의 숫자 타입만 존재
    - String : 문자열
    - Boolean : 논리적 true & false
    - undefined : var 키워드로 선언된 변수에 암묵적으로 할당되는 값은
    - null : 값이 없다는 것을 의도적으로 명시할 때 사용하는 값
    - symbol : ES6에서 추가된 7번째 타입
    - object : 객체, 함수, 배열 등
